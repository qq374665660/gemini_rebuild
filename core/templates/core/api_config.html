使用说明

了解如何配置和使用AI服务

配置步骤

  1. 选择要配置的AI服务类型
  2. 输入对应的API密钥
  3. 点击"保存配置"按钮
  4. 使用"测试连接"验证配置是否正确
  5. 启用服务开始使用AI分析功能

注意事项

  • API密钥将被安全加密存储
  • 建议定期测试连接以确保服务可用
  • 同一服务类型只能配置一个API密钥
  • 禁用的服务不会被用于AI分析
